# 🏦 Alke Wallet

Billetera digital desarrollada con Django como proyecto final del Módulo 7 - Desarrollo Web con Django.

## 📋 Descripción

Alke Wallet es una aplicación web que permite gestionar clientes, cuentas y transacciones financieras de forma segura y sencilla, utilizando el ORM de Django y buenas prácticas de desarrollo.

## 🛠️ Tecnologías utilizadas

- Python 3.14
- Django 6.0.4
- SQLite (desarrollo)
- HTML/CSS
- Git y GitHub
